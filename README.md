# baiinfo-ui 组件库

### 组件文档
https://github.com/hgr-baiinfo/baiinfo-ui
### 快速开始

> 在 main.js 中引入组件库

```javascript
// 全部引入
import 'baiinfo-ui/dist/css/index.css'
import BUI from "baiinfo-ui";
Vue.use(BUI);

// 按需引入
import 'baiinfo-ui/dist/css/demo.css'
import { Demo } from "baiinfo-ui";
Vue.use(Demo);

```
